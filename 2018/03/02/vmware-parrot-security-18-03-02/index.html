VMware 安装 Parrot Security 3.11

0x2h
2018-03-02
hacker, linux, parrto security, vmware

前言

这些天玩linux上瘾了，上午在油管看到 Parrot Security 立马就试了试，官网写着是基于 Debian 10，但是debian 10 好像还没出来吧。。。不瞎比比了，开工

正文

  • 首先到官网下载镜像
  • 选择 Security 下载
  • 根据自己系统选择 64位 或者 32位 下载
  • 接下来 打开VMware - 新建虚拟机 - 典型 - 稍后安装系统
  • 这里要求选择版本，选择 Debian 9.x，多少位看自己下载的镜像是多少位的
  • 虚拟机名称和安装的位置自己设置
  • 磁盘容量最低20GB，安装系统就要12G(记不太清楚了)，然后自己就没多少内存了，我固态只有256，所以我选择 30GB，当然是越大越好
  • 然后点自定义硬件
  • 设置内存，大小看自己情况
  • 处理器自己设置
  • 然后在 新CD/DVD 里使用自己下载的 ISO文件
  • 其他的设置自己随意，然后点 关闭 - 完成。

    接下来就是安装了

  • 启动虚拟机。。在这里选择 Install 然后回车
  • 然后选择 Install with GTK GUI 回车
  • 选择语言 这不说了，这里选择 是
  • 接着就是选择国家，然后设置 root 密码
  • 然后就是新建账户以及设置账户密码，这个就自己弄

  • 然后磁盘分区，没特殊要求，一路回车即可
  • 然后这里询问改动的磁盘是否写入，选择是。接下来就是漫长的安装系统

  • 在安装快结束的时候会出来这个问题，选择 是，然后选择 /dev/sda 就可以了，一直等到安装结束

  • 安装完成，接着会重启（不是重启电脑）

    接下来是设置系统

  • 直接回车
  • 登陆，看着这窗口要瞎眼，接下来安装 vmware tools
  • 打开terminal

  • 先更新一波软件源

  • 接着安装 open-vm-tools-desktop

  • 安装完成之后重启

  • 然后看到 VMware 的菜单，选择 查看 - 立即适应客户机。然后你就可以看到大窗口了
